## 🚀 **Quick Start for Judges**

### **Local Development:**
```bash
# Install dependencies
npm install

# Start development server
npm start

# Open in browser
# http://localhost:3000
```

### **Build for Production:**
```bash
# Create optimized build
npm run build

# Serve build files
npx serve -s build
```
